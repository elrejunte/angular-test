Ejercicio Angular

  1. Crear un componente que reciba un pais y muestre el nombre del pais en pantalla (En stackbliz se puede crear con boton derecho -> Angular Generator)

  2. Usando el servicio CountryService mostrar el listado de paises con el componente creado

  3. Agregar un evento al componente creado para que al clickearlo muestre en el componente padre ('app.component') el id del item clickeado

  4. Tipar el servicio de paises

  5. Ordenar los paises a mostrar por cantidad de letras, de menor a mayor. Ej: [Cuba,Fiji,Angola,Montenegro,Central African Republic], Sin utilizar ninguna libreria.

  6. Aplicarle estilos visuales al listado y a los elementos

Tips

  • Puedes googlear, no pasa nada!

  • el punto 4, 5 y 6 no son obligatorios, pero suman puntos extra
